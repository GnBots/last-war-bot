# Last War Bot Features

GnBots Last War Bot is a cloud automation tool designed to handle repetitive Last War: Survival tasks.

## Auto Join Rally

Automatically join alliance rallies without manually watching your phone all day.

Benefits:

- Run 24/7
- Never miss alliance rallies
- Earn more rewards
- Save time

## Last War Map Scanner

The map scanner searches the server map automatically.

Use it to help find:

- Players
- Zombies
- Targets
- Important map locations

## Gold Zombie Finder

Gold Zombies provide valuable rewards but manually searching the map takes time.

GnBots can scan and help locate Gold Zombies automatically.

## Auto Shield Protection

Monitor your base and react to incoming attacks.

Protect your progress when you are away.

## Daily Automation

Automate repetitive daily tasks:

- Resource collection
- Missions
- Events
- Upgrades
- Training
