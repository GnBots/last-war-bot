# Last War Bot Setup Guide

This Last War Bot setup guide explains how to start using GnBots for Last War: Survival automation.

Unlike traditional GitHub bots, GnBots does not require Python scripts, emulators, APK files, or a PC running 24/7. GnBots is a cloud-based Last War bot built to automate the repetitive parts of the game from an online dashboard.

## What You Need

To use GnBots Last War Bot, you need:

* A Last War: Survival account
* Access to your Last War account email or login method
* An active GnBots account or free trial
* A web browser on any device

You do not need:

* A PC
* An emulator
* Python
* GitHub code
* Downloads
* APK mods

After selecting your features, start the bot from the GnBots dashboard.
